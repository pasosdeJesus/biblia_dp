# WEB Classic (Yahweh Edition) — USFX

**Fuente:** [World English Bible Classic](https://ebible.org/eng-web/)

**Descargado de:** [https://ebible.org/usfx/](https://ebible.org/usfx/)

**Archivo principal:** `eng-web_usfx.xml`

**Descripción:** Texto completo de la World English Bible (WEB) en formato USFX, edición "Classic" que translitera el Tetragrámaton como "Yahweh" (a diferencia de la edición estándar que usa "LORD").

**Licencia:** Dominio público (Public Domain). "World English Bible" es una marca registrada de eBible.org.

**Versión:** 2020 stable text edition (generado el 23 Abr 2026 con Haiola por eBible.org).

**Uso en este proyecto:**
- Fuente base en inglés para la traducción del Antiguo Testamento al español.
- Cada palabra tiene marcado `<w s="HXXXX">` con números Strong hebreos.
- Se convirtió a GBFXML mediante `herram/usfx2gbfxml.mjs`.
